Plugin for adding a rotating image graphic across the header of your site

== Description ==

This plugin uses a light weight javascript file along with the jquery-1.4.1.min.js library to create a cycling image gallery with fading transitions.
It also has and editor under the seeting tab that allows you to upload images from there as well you may delete images from there. 

== Installation ==

1. Upload `looping_image.zip` to the `/wp-content/plugins/` directory
2. Unzip `looping_iimage.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Place `<?php echo looping_image(); ?>` in your templates where ever you'd like it to be shown
5. Add images using the Looping Image tab under settings
6. Check the live site
7. Style it how ever you like, enjoy!

== Frequently Asked Questions ==

= Can I add more that one? =

As of right now Looping Image Plugin can only be once per site, we are working on making it more scalable.
